PAGINA EN WORD PARA DIFERENCIAR ENTRE HTML Y WORD

DIFERENCIAS.

Al guardar un documento de Microsoft Word como una página Web (en el menú archivo , haga clic en Guardar como página Web), Word convierte automáticamente la página HTML (lenguaje de marcado de hipertexto), el lenguaje utilizado por los exploradores Web para leer páginas Web.

Dado que Word muestra la página Web similar a la forma en que la página se mostrará en un explorador Web (Microsoft Internet Explorer), ciertos tipos de formato y otros elementos que no son compatibles mediante HTML o el entorno de creación de páginas de Web, no se muestran en Word (o en los exploradores Web).

MAS INFORMACION

Cuando guarda un documento de Word 2000 como página Web, Word le avisa con el mensaje "Algunas de las características de este documento no son compatibles con exploradores Web" y muestra un resumen de las características que no son compatibles y cómo aparecen en un explorador Web.

Se enumeran los elementos que Word cambia o quita cuando guarda un archivo como una página Web.

  • Formato de caracteres
  • Formato de parrafo
  • Diseño de pagina
  • Edicion
  • Formato de grafico
  • Formato de texto asiatico
  • Formato de tabla
  • Formato de pagina web
